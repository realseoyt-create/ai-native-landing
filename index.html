Test기술팀의 AI 전환은 4개의 축으로 동시에 진행됩니다. 각 과제는 독립적으로 실행되면서도 서로 연결되어 시너지를 만듭니다.

AXIS 01
AI Native 전환
팀원 스스로 AI를 활용해서 일하게

팀원이 각자의 업무 방식에 맞는 AI 에이전트를 직접 만들고 운영합니다. 환경 설정을 한 번에 완료할 수 있는 하네스 플러그인으로 진입 장벽을 없애고, 오픈코드 기반으로 보안 이슈 없이 사내 환경에서 바로 쓸 수 있습니다.

  • ▸환경 설정 1회 → 즉시 사용 가능한 하네스 플러그인
  • ▸오픈코드 기반 구현, 보안 이슈 제거
  • ▸페르소나 에이전트 자율 개발
  • ▸Data Lake MCP 연동, 개인 활용 사례 팀 자산화
  • ▸DevOps 적용 — 클라우드 서버 배포 지원 · 안전한 배포 환경 구축
진행 중팀원 역량 강화 · 팀 전체 확산

AXIS 01 · AI NATIVE 🧑‍💻

팀원 스스로 AI를 활용해서

  • 👥대상: Test기술팀 전 팀원

  • 📦결과물: 페르소나 에이전트, 팀 하네스 기여

Core Concept · AI Native

하네스(Harness)란

하네스는 AI가 일을 잘 할 수 있도록 설계된 작업 환경 패키지입니다. 오픈코드 기반으로 구성해 보안 이슈 없이 사내에서 운용하며, 플러그인처럼 배포해 누구나 환경 설정 한 번으로 즉시 사용할 수 있습니다.

harness / 6가지 구성 레이어
LAYER 01

컨텍스트

AI가 이 프로젝트와 팀의 규칙을 이해하도록 설계된 지식 레이어.
CLAUDE.md

LAYER 02

스킬(Skills)

반복되는 워크플로우를 명령어 하나로 실행하는 재사용 가능한 레시피.
/skill-name

LAYER 03

훅(Hooks)

파일 수정, 명령 실행 등 이벤트에 자동으로 반응하는 트리거.
PreToolUse / PostToolUse

LAYER 04

메모리(Memory)

대화가 끊겨도 AI가 팀과 프로젝트를 기억하는 영속 기억 레이어.
memory/*.md

LAYER 05

도구 연동(MCP)

슬랙, 텔레그램, GitHub 등 외부 도구를 AI가 직접 다루도록 연결.
MCP Servers

LAYER 06

에이전트 오케스트레이션

복잡한 작업을 여러 에이전트가 분담해 병렬로 처리하는 구조.
fork / subagent

사람이 깜빡할 수 있는 단계를 훅이 대신합니다. PreToolUse / Stop

🚀

DevOps — 안전한 클라우드 배포 환경

AI가 배포 파이프라인에 직접 참여합니다. 코드 변경부터 클라우드 서버 배포까지 검증·승인·롤백 단계를 자동화해 사람의 실수를 줄이고 배포 안전성을 높입니다.

CI/CD 자동화 · 배포 전 사전 검증 · 이상 감지 자동 롤백

AXIS 02 · Forward Deployed Engineer

FDE 과제 밀착

TSS가 과제별로 밀착 투입되어 처음부터 끝까지 함께 몰입합니다. 그 경험을 팀 전체로 확산합니다.

FDE란 무엇인가

Forward Deployed Engineer

FDE는 실리콘밸리에서 시작된 개념입니다. 기술 엔지니어가 과제에 처음부터 끝까지 밀착해서 실제 문제를 함께 해결하고 성과를 만드는 방식입니다. Test기술팀에서는 TSS 엔지니어가 과제별로 몰입 투입되어 반복 업무를 자동화하고, 그 경험을 팀 전체에 전파합니다.

일반 지원 방식 : 요청이 오면 대응 → 해결해주고 끝 (단발성, 수동적)
FDE 방식 : 과제에 밀착 · 함께 몰입 → 처음부터 끝까지 책임지고 완결 (지속적, 능동적)
확산 방식 : 개별 해결에 그치지 않고 → 재현 가능한 하네스로 패키징 → 팀 전체 배포
STEP 01

과제 선정

자동화 효과가 높은 과제를 선정합니다. Excel 의존도, 반복 업무 비율, 팀 체감 Pain 기준.

STEP 02

TSS 밀착 투입

TSS 엔지니어가 과제에 밀착합니다. 요청 대기가 아니라 함께 몰입해서 Pain Point를 선제 파악합니다.

STEP 03

함께 완결

반복 업무를 자동화하고 E2E Workflow를 개선합니다. 중간에 끊지 않고 끝까지 책임집니다.

STEP 04

팀 전체 확산

성공 사례를 재현 가능한 하네스로 패키징해 Test기술팀 전체로 전파합니다.
